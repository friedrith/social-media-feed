# Social Media Feed

## Getting started

```bash
yarn

yarn dev
```

## Technologies

This repository is based on:

- next.js (build: ssr, fast)
- react.js
- redux-toolkit (less code than redux)
- tailwind+[tailwindui](https://tailwindui.com/)+[flowbite](https://flowbite.com/) (style+components)
- prettier + style (code style)
- i18next (location)
- jest (test)
- simple-git-hooks
- commitlint
- github actions
